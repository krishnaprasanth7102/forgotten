# **App Name**: FORGOTTEN: Developer Arsenal

## Core Features:

- Boss Package Download: Download complete boss packages including models, AI behavior, combat design, lore, and engine compatibility.
- Integration Mode: Access dedicated instruction views for each boss, with step-by-step integration guides for Unity, Unreal Engine, and Godot.
- Open-Source Contribution: Fork, modify, and submit upgrades or new bosses to the platform, fostering a collaborative environment.
- High-Impact Boss Preview: Preview bosses with striking visuals and key stats upon selection, ensuring developers are armed with essential information.
- Download as Arming: The download interaction will incorporate visual feedback cues to feel like arming a weapon

## Style Guidelines:

- Primary color: White (#FFFFFF) for structure and text, providing a clean, technical feel against the black environment.
- Background color: Black (#000000) for the environment, creating a sense of a classified, high-tech control interface.
- Accent color: Red (#FF0000) exclusively for action, danger, and interactive elements, highlighting actionable items and critical information.
- Font: 'Space Grotesk', a bold, modern sans-serif, for a strong visual hierarchy and contemporary feel. Note: currently only Google Fonts are supported.
- Single continuous flow layout with no traditional pages, sharp lines, asymmetry, and sections that bleed into each other.
- Mechanical and controlled motion for interactions, avoiding playful animations. Subtle system feedback, such as pulse and scan lines, on downloads.
